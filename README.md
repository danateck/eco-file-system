# Eco-Files (React wrapper)

This React project preserves your original **logic, keywords, and styles**.
- Original DOM from `index.html` is injected via `src/content/body.html`.
- Original scripts are loaded in order by `src/App.jsx`.
- All original assets live under `public/` with the same paths.

the app is at: eco-docs.netlify.app

## Run
```bash

install node 18+
open terminal and run:
npm install
npm run build
npm run preview 
```
